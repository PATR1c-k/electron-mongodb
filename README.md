# Implementation

1. First clone the project.
``` bash
git clone https://github.com/PATR1c-k/electron-mongodb.git
```

2. make a globals.js in directory (root).
## globals.js
```javascript
const URI = "";
const DB_NAME = "employeeData";

module.exports = {
  URI,
  DB_NAME,
};
```
Enter your string for mongoDB connection.
You are ready to go!
